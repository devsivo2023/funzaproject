Craft Certificate in Information Communication Technology

This course is intended to equip the trainee with knowledge, skills and attitudes which are responsive to the challenging needs in the computing industry. The course will allow the graduate to work in a salaried employment or self-employment. To achieve this, units such as Entrepreneurship, Communication Skills and Life Skills have been included. During the course the trainee shall be exposed to the following core areas: a) Computer maintenance and user support b) programming c) Computer Applications d) Operating systems.

  • Life Skills
  • Introduction to ICT
  • Computer Applications I
